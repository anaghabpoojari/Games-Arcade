# Games Arcade  
A full-stack **Mini Game Arcade** web application featuring multiple classic games with **user authentication** using **MongoDB**. Users can register, log in, and play games in an interactive dashboard.

## Features
- User authentication (Register / Login)
- Secure password storage
- Protected routes (only logged-in users can play)
- Multiple classic mini-games
- Clean and responsive UI
